Welcome to Resume Filter Application


Our ultimate motive is to provide a better solution to the current hiring process using Text and Psychological analysis of the applicant. Giving a better view of applicant’s psychological understanding to the recruiter.

  1. Applicants can Sign up, log in and fill the resume application form of companies.
  2. Recruiters can further view the resume application form data and sort them according to their culture of the company.
  3. Administrators can check the list of applicants, applicant’s forms and recruiters.

API's Used:

  1. Personality Prediction API using CatBoost Classifier algoritm and Flask
  2. Sentiment Prediction API using Sequential Neural Network algoritm and Flask

Future of the project:

We could use AWS Cloud Services to further improve my application,
  1. Amazon Simple Storage Service (Amazon S3) for better scalability, data availability, security, and performance.
  2. AWS EC2 for better computational resouces required by our API's.
  3. Amazon Sagemaker to better train Machine Learning Algorithms and to create end points to communicate with them.
